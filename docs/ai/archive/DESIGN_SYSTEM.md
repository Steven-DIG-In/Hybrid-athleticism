# DESIGN SYSTEM & AESTHETICS

## Vision
The application must embody "luxury high-end gloss" fused with the raw, uncompromising awe of extreme nature. It is a premium tool for users who train not just for the gym, but for life. When they stare into a bottomless pit, they don't retreat; they ask for their rope.

### Core References
- **Extreme Nature & Horizon:** Misty canyons, bottomless pits, expansive, vast landscapes that inspire awe and a slight sense of danger. 
- **Audemars Piguet:** High-end, precise, bold storytelling, mechanical complexity wrapped in beautiful housing.
- **Superlist:** Ultra-clean, immersive, minimalist productivity.
- **Spotify Design:** Vibrant, tasteful use of scroll-triggered animations and deep, smooth micro-interactions.
- **Ellipsus & Seed.com:** Elegant, editorially-driven typography, bold engaging visuals, and memorable functional storytelling.

## 1. UI/UX Principles
- **Vast & Atmospheric:** Incorporate atmospheric depth. Deep blacks should feel like an endless night sky; backgrounds should hint at misty, rugged textures rather than flat color.
- **Ultra-Clean & Minimalist:** Remove cognitive load. Space is a premium asset. Vast negative space represents the expansive horizons the elite train for.
- **Immersive & Interactive:** Use subtle micro-animations (fades, smooth transitions, scroll-triggers). The app should feel "alive" and responsive to every touch.
- **Editorial Typography:** Typography should feel like a high-end magazine. High contrast between headers and body text. 

## 2. Data Visualization
- **The Core User:** The user loves their training data. Data cannot be hidden, but it must be presented beautifully.
- **Atmospheric Charts:** Use high-end charting libraries customized to look sleek and glowing against the rugged, atmospheric backgrounds. 
- **Beautiful Data Grids:** When diving deep into volume, RIR, tonnage, and maxes, the tables and grids must be beautiful—properly aligned, impeccably spaced, and highly legible.

## 3. The "Stitch / Antigravity" Directive
When generating new components via Stitch or Antigravity:
- Avoid generic colors (plain red, blue, green). Use curated, harmonious color palettes (e.g., sleek dark modes with metallic/neon accents or ultra-crisp light modes with slate/navy contrast).
- Ensure modern typography (e.g., Inter, Roboto, Outfit).
- Never use placeholder imagery; generate or use premium placeholder assets.
- If it doesn't WOW the user at first glance, it is a failed component.
