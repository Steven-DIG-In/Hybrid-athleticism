# Hybrid Athleticism

AI-powered training platform for hybrid athletes. Next.js + Supabase + Anthropic Claude.

## AI Development

The single source of truth for this project is:

**`docs/ai/MASTER_PROMPT.md`**

All prior prompt files have been archived to `docs/ai/archive/` and should not be referenced.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Tech Stack

- Next.js (App Router), React, TypeScript
- Tailwind CSS, Framer Motion
- Supabase (PostgreSQL + Auth + RLS)
- Anthropic Claude API
